# Web Tasarım Dersi Uygulamaları

## Açıklama
Bu repository, Ondokuz Mayıs Üniversitesi Bilgisayar Programcılığı 1. sınıf Web Tasarım dersinde geliştirdiğim uygulama projelerini içermektedir.

## İçerik
Bu depo, web tasarım öğrenme sürecimdeki ilerlememi göstermektedir:

### Öğrenme Evrimi:
- **İlk Haftalar:** Temel HTML sayfaları ve ders notları uygulamaları
- **İlerleyen Haftalar:** CSS entegrasyonlu modern web sayfaları
- **Son Haftalar:** Tamamen responsive ve estetik projeler

### Teknolojiler:
- HTML5
- CSS3
- Responsive Tasarım

## Not
Bu projeler akademik çalışmalarımın bir parçası olup, web tasarım alanındaki gelişim sürecimi yansıtmaktadır.
