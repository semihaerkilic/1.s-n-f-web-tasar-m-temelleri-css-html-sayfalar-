CSS çalışması

CSS - Cascading Style Sheets kelimelerinin başharflerinin kısaltmasıdır. Anlamı birbirini takip eden sayfalarıdır.

CSS Neden Kullanılır?

Html sayfasının omurgasını oluştururken (metinler, resimler, listeler, linkler, tablolar, formlar...) Css web sayfasının dizayn sayfasını elementlerini oluşturur.

html etiketlerini dizayn etiketlerinden ayırarak sayfanın sadeleşmesini sağlar ve kullanım kolaylığı sağlar.

CSS Web sayfasına 3 şekilde eklenir:

1.Yerel CSS Ekleme:

Css etiketleri direkt olarak HTML etiketlerinin içerisine yazılır ve sadece bu etiketi etkiler.

2.Genel CSS Ekleme:

CSS etiketleri sayfanın head bölümüne yazılır ve sayfanın tamamını etkiler.

3. Harici CSS Ekleme:

Css etiketleri farklı bir sayfaya yazılır, .css uzantılı olarak kaydedilir. Link verildiği tim web sayfalarını etkiler.

CSS web sayfalarında 3 şekilde yazılır:

1. Tag (Etiket Stili) Stili Yazma:

CSS etiketleri direkt olarak html etiketine yazılır.

Resim Ekleyelim

2. Class (Sınıf Stili) Stili Yapma:

css ile biçimlendirilecek olan html etiketine bir sınıf atanır(class) css sayfasında .classismi yazılarak css biçimlendirilmesi yapılır.Birden fazla html etiketine aynı class ismi verilebilir.

ID (kimlik) stil Yazma:

css ile biçimlendirilecek olan html etiketine bir id atanır(id) css sayfasında .idismi yazılarak css biçimlendirilmesi yapılır.Birden fazla html etiketine aynı id ismi verilemez.
